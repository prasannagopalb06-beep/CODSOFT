##  Task Description

Build a simple chatbot that responds to user inputs based on predefined rules.  
The chatbot identifies user queries using **pattern matching (Regex)** and returns appropriate responses.

This task helps in understanding:
- Basic Natural Language Processing (NLP)
- Conversation flow
- Rule-based decision logic



##  Technologies Used

- Python  
- Regular Expressions (`re` module)  
- Gradio  
- Datetime module  
- Google Colab  



##  How It Works

- User enters a message
- Input is converted to lowercase
- Regex patterns are matched against predefined rules
- If a match is found, a response is returned
- If no match is found, a fallback response is shown
